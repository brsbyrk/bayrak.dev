# bayrak.dev Blog

Welcome to the source code for my personal blog, [bayrak.dev](https://bayrak.dev). Built with [Astro](https://astro.build) and using this [theme](https://github.com/nicdun/astro-tech-blog), this site is where I share insights, projects, and tutorials on tech topics and my personal journey.

## Features

- **Astro Framework**: For a fast, optimized static site.
- **Markdown Blog Posts**: Easy-to-write and read posts with Markdown.
- **Minimalist Design**: A clean and simple layout to focus on content.

## Structure

- **`src`**: Contains all site content, components, and layouts.
- **`public`**: Static assets like images and icons.
- **`posts`**: Blog posts written in Markdown.
- **`layouts`**: Custom layouts for pages and posts.
